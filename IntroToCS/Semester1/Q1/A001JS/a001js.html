Script Tags and Comments

A001JS

JavaScript is coded using the < script > and < /script >: tags. It's similar to creating the % lt; style > tags in the head section. 
            
Example

            
            

                < script >
                // your code goes here!
                < /script >

Notice the double slashes within the script tag? The double slashes // allow you to comment your code which will become important over time.

For this assignment include the script tag and several comments that include the following information:

  • JavaScript is case sensitive (capitalization matters!)
  • JavaScript is minimum white-space sensitive (enters matter!)
  • JavaScript is sensitive to apostrophes (or single quotes)
  • JavaScript lines should end with a semi-colon
           

                < script >
                    
                        // Capitlization matters
                        // Minimumwhite space matters
                        // Apostrophes and single quotezs matter
                        // All code lines should end with a semi-colon
                    
                < /script >
